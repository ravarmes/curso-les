Fase de construção do projeto. Toda codificação do sistema é realizada nesta fase.

 

 

 

 

 

09 Atividade: Implementar

Finalidade

  • Codificar as classes necessárias para implementação das funcionalidades de processos de negócio, manutenção de cadastros e listagens.
  • Observação: como artefatos de saída desta atividade serão gerados dois projetos:
    • Back end: Node e Sequelize
    • Front end: React.

Artefatos de Entrada:

  • Código Fonte - Backend - Classes de Persistência (CB1)

Artefatos de Saída:

  • Código Fonte - Backend (CB2)
  • Código Fonte - Frontend (CF1)
  • SCV-Postman

Papel: Programador

Ferramentas: Visual Studio Code; Postman;

Implementar o Back End (Web Service REST):

  1. Abrir uma ferramenta para o ambiente de programação (Visual Studio Code);
  2. Implementar as novas classes no projeto já iniciado;
  3. Compactar o projeto e salvar com o seguinte nome: <Sigla do Projeto>-backend_02.rar

 

Implementar o Front End (tecnologia de sua escolha):

  1. Abrir uma ferramenta para o ambiente de programação;
  2. Criar um projeto na tecnologia de sua preferência (React, JavaFX, Ionic, PHP, etc);
  3. Compactar o projeto e salvar com o seguinte nome: <Sigla do Projeto>-frontend_01.rar

Detalhar as requisições ao Back End no Postman:

  1. Abrir uma ferramenta Postman;
  2. Detalhar todas as requisições das transações a serem consideradas no trabalho. Estas requisições devem estar completas (incluindo parâmetros, quando necessário);
  3. Compactar o projeto e salvar com o seguinte nome: <Sigla do Projeto>-Postman.json
